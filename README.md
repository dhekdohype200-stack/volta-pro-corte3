# Volta Pro Corte 💈

O **Volta Pro Corte** é um SaaS premium desenvolvido especificamente para barbeiros que desejam aumentar seu faturamento recuperando clientes que não retornam há algum tempo. Através de uma interface intuitiva e focada em conversão, o barbeiro consegue identificar clientes inativos e entrar em contato diretamente via WhatsApp com apenas um clique.

## 🚀 Funcionalidades

- **Dashboard Inteligente:** Visualize rapidamente quantos clientes estão "sumidos" e quanto faturamento você pode recuperar.
- **Gestão de Clientes:** Lista completa de clientes com status de retorno (Pronto para voltar, Em dia, etc.).
- **Agenda Integrada:** Sistema de agendamento simples e eficiente.
- **Link de Agendamento Público:** Compartilhe seu link exclusivo no Instagram ou WhatsApp para que seus clientes agendem sozinhos.
- **Recuperação via WhatsApp:** Botões de ação rápida que abrem o WhatsApp com mensagens personalizadas para cada cliente.
- **Onboarding Guiado:** Configuração inicial rápida para começar a usar em menos de 1 minuto.
- **Design Premium:** Interface escura, moderna e totalmente responsiva (Mobile-First).

## 🛠️ Tecnologias

- **Frontend:** React 19, Vite, Tailwind CSS 4, Framer Motion, Lucide React.
- **Backend:** Node.js, Express.
- **Banco de Dados:** SQLite (Better-SQLite3).
- **Estilização:** Tailwind CSS com design system personalizado.
- **Ícones:** Lucide React.

## 📦 Como rodar o projeto

### Pré-requisitos

- Node.js (v18 ou superior)
- npm ou yarn

### Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/volta-pro-corte.git
   cd volta-pro-corte
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Configure as variáveis de ambiente:
   Crie um arquivo `.env` na raiz do projeto (use o `.env.example` como base).

4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

5. Acesse no navegador:
   `http://localhost:3000`

## 🏗️ Estrutura do Projeto

- `/src`: Contém todo o código fonte do frontend (React).
- `/server.ts`: Servidor Express que gerencia a API e o banco de dados.
- `/database.db`: Banco de dados SQLite local (gerado automaticamente).
- `/public`: Arquivos estáticos.
